# 🚀 CreativeHub

**CreativeHub** is a fully responsive **one-page corporate website** developed using **HTML, CSS, and Vanilla JavaScript**.

This project represents a **design agency scenario**, featuring essential business sections such as **About, Services, Location, and Contact**, built with a clean structure and modern UI approach.

🔗 **Live Demo:** https://creativehubbb.web.app/


---

## 🎯 Project Purpose

CreativeHub was developed to simulate a **real-world agency website**, focusing on both **design presentation** and **frontend architecture**.

The project aims to:

* 🏢 Build a complete **corporate one-page website**
* 🎨 Present a modern and professional agency identity
* ⚡ Improve **Vanilla JavaScript** skills without frameworks
* 🔄 Use **JSON-based dynamic content rendering**
* 🧱 Create a scalable and maintainable project structure

---

## 🧩 Core Features

* 🏠 **Full One-Page Structure**  
  All essential sections (Home, About, Services, Location, Contact) are presented on a single page

* 🧾 **About Section**  
  Brief introduction of the agency and its vision

* 🛠️ **Services Section**  
  Clear presentation of offered services with structured layout

* 📍 **Location Section**  
  Company location information integrated into the page

* 📞 **Contact Section**  
  Simple and accessible communication area

* ⚡ **Dynamic Content (JS + JSON)**  
  Content is rendered dynamically for flexibility and scalability

* 📱 **Responsive Design**  
  Optimized for mobile, tablet, and desktop devices

* 🎨 **Modern UI Design**  
  Clean and minimal interface inspired by real agency websites

## 🛠️ Technologies & Tools

* **HTML5** – Semantic structure  
* **CSS3** – Responsive and modern styling  
* **Vanilla JavaScript** – Application logic
